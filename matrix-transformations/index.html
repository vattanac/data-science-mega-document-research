📖 What Are Matrix Transformations?

A matrix transformation is a linear function that maps vectors from one space to another using matrix multiplication. Geometrically, it describes how points in space rotate, scale, shear, or reflect.

For a 2D point x = [x, y]ᵀ and a 2×2 matrix M, the transformation is: x' = M · x

Key Property: Matrix transformations preserve lines and the origin. They're fundamental in computer graphics, data analysis, and machine learning.

Linear Maps & Geometric Interpretation

Linear Maps

Functions f(x) where f(ax + by) = a·f(x) + b·f(y). Matrix transformations are linear maps.

Basis Vectors

A matrix's columns represent where the standard basis vectors [1,0] and [0,1] map to.

Determinant

The determinant tells us the area scaling factor and whether orientation is preserved.

Eigenvalues/Eigenvectors

Special vectors that only scale (don't rotate) under the transformation.

🎨 2D Transformation Types

Rotation

Rotates a point counterclockwise by angle θ (in degrees).

R(θ) = [ cos(θ) -sin(θ) ]
        [ sin(θ) cos(θ) ]

Scaling

Scales points by factors sx and sy along x and y axes.

S(sx, sy) = [ sx 0 ]
            [ 0 sy ]

Shearing

Shears points along one axis by a shear factor h.

Shear(h) = [ 1 h ]
               [ 0 1 ]

Reflection

Reflects points across the x-axis, y-axis, or the line y=x.

Reflect X = [ 1 0 ]   Reflect Y = [ -1 0 ]   Reflect Y=X = [ 0 1 ]
                 [ 0 -1 ]                   [ 0 1 ]                   [ 1 0 ]

🔗 Composition of Transformations

When you apply multiple transformations in sequence, the overall effect is a single matrix obtained by multiplying the transformation matrices.

If you rotate first, then scale, the combined transformation is: M_combined = M_scale · M_rotate

Important: Matrix multiplication is not commutative. The order matters! Rotating then scaling gives a different result than scaling then rotating.
Example: Rotation(45°) then Scale(1.5, 1.5):

M_combined = [ 1.5 0 ] · [ cos(45°) -sin(45°) ] = [ 1.06 -1.06 ]
                    [ 0 1.5 ] [ sin(45°) cos(45°) ] [ 1.06 1.06 ]

🚀 Applications

Computer Graphics

Matrix transformations are the foundation of 3D graphics rendering. Every rotation, scaling, and projection in video games and animation software uses matrices. GPUs are optimized for matrix operations.

Neural Networks

Each layer of a neural network applies a linear transformation (via matrix multiplication) followed by a non-linear activation. Deep learning is fundamentally about composing transformations.

Principal Component Analysis (PCA)

PCA uses eigendecomposition of the covariance matrix to find directions of maximum variance in data. The eigenvectors form a transformation to lower dimensions while preserving information.

Image Processing

Rotating, scaling, and warping images all use transformation matrices. Convolution operations in modern computer vision are also matrix operations.

🎬 Animation

Interpolating between transformation matrices creates smooth animations.

📊 Data Transform

Whitening and normalizing data uses matrix transformations.

🔐 Cryptography

Hill cipher uses matrix operations for encryption.

🌐 Robotics

Robot arm kinematics uses transformation matrices.
