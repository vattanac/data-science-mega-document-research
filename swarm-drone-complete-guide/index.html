Suggested Learning Path

Week 1-2

Start with Part 1: Mathematics

Build the mathematical intuition. Focus on vectors, the Kalman filter, and graph Laplacian — these three concepts appear everywhere in swarm systems.

Week 3-4

Move to Part 2: Algorithms

Implement Reynolds' boids in Python. Then add PID control and A* path planning. The interactive demos help you build intuition.

Week 5-6

Explore Part 3: Software

Set up ROS2 and Gazebo. Run your first multi-drone simulation. Connect your Python algorithms to simulated drones.

Week 7-8

Study Part 4: Hardware

Understand every component. Order parts for your first drone build. Start with a single drone before attempting a swarm.

Week 9-12

Execute Part 5: Integration

Follow the V-model pipeline. Test in simulation first, then SITL, then single drone, then small swarm. Never skip testing phases.
